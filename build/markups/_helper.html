.helper

Нельзя изменить в дальнейшем

Хелперы вывводятся после инпута, а не внутри него. Это нужно для большей универсальности, чтобы можно их было использовать с любыми компонентами:

Random element
Изменения сохраняются автоматически



.helper--error

Алсо используется для вывода ошибок:

Пароль слишком короткий
Пароль должен содержать спецсимвол
Пароль должно быть сложно угадать: даты, имена, и номера телефонов и прочую открытую информацию нельзя использовать в качестве пароля
Новый пароль не может быть идентичен старому
