Разметка внутри

Предполагается, что внутри будут небольшие тексты, без переносов строк. По умолчанию переносы запрещены. Если вдруг нужен многострочный текст, то проставляем <br>`ы вручную.




Показ бабла

Клик по .bubble-handler ищет и показывает соответствующий бабл. Соответствующий этот тот, который находится внутри .bubble-context вместе с тем .bubble-handler на который кликнули. Это нужно главным образом для мобилок без ховера.

Ещё бабл показывается обычным CSS-ным наведением на .bubble-context
